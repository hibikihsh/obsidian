## 自己 PR

**1. 設計から実装まで行う 0→1 での Web システム構築**

各プロジェクトにおいて、設計から実装、テスト、運用/保守までを一貫して担当し、0→1 での Web システム構築のスキルを磨きました。開発で特に注意した点としては、リリース後に発生する問題を最小限にすることです。Pull Request 時に linter や単体テストを実行するよう設定し、 各ブランチにマージすると, 検証・本番環境に自動デプロイされる仕組みを構築しました。また、チームで開発する際においても、プロダクトに関する疑問点や改善点は必ず委託に相談・提案し、各機能をどの粒度で実装するかを考えて納期を厳守するよう心がけています。

**2. 心理学の手法を取り入れた UI の設計と実装**

大学院時代に学んだ心理学の手法を UI デザインに取り入れて、よりユーザービリティの高いシステムの構築を目指しています。特に UI の実装で気をつけている点として、ユーザーのメンタルモデルを損わないような設計を心がけています。また、ピーク・エンドの法則などに基づいてユーザーがどの機能でもっとも肯定的な感情を抱くのかなどを考えながら UI の構築を行っています。
