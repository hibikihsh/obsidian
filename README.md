# Obsidian プロジェクト構造

## ディレクトリ構成

```
Obsidian/
├── .obsidian/                          # Obsidian設定ディレクトリ
│   ├── themes/Minimal/                 # テーマ設定
├── 00_Bases/                           # ベースノート
├── 01_IndexNotes/                      # インデックスノート
├── 02_PermanentNotes/                  # 永続ノート
├── 03_LiteratureNotes/                 # 文献ノート
├── 04_FleetingNotes/                   # フリーノート
├── 05_Bookmarks/                       # ブックマーク
├── Documents/                          # ドキュメント
├── Prompts/                            # プロンプト
└── README.md
```

## 各ディレクトリの説明

| ディレクトリ                 | 説明                      |
| ---------------------- | ----------------------- |
| **00_Bases**           | Zettelkasten方式のベースノート定義 |
| **01_IndexNotes**      | ノートのインデックス・目次           |
| **02_PermanentNotes**  | 永続的な知識ノート（Zettelkasten） |
| **03_LiteratureNotes** | 文献・参考資料からの抽出ノート         |
| **04_FleetingNotes**   | 一時的なメモ・フリーノート           |
| **05_Bookmarks**       | ブックマーク・参考リンク集           |
| **Documents**          | 仕事に関する関連ドキュメント          |
| **Prompts**            | カスタムプロンプト               |

## 特徴

- Zettelkasten方式のノート管理システム
- キャリア・エンジニアリング関連の知識管理
- ブックマークと参考資料の整理
- カスタムプロンプトの保存
